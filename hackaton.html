We split the training set to train and check our model

library(caTools) # install it first in the console
## Warning: package 'caTools' was built under R version 3.6.3
set.seed(703185)
split = sample.split(data_train, SplitRatio = 0.70)
training_set = subset(data_train, split == TRUE)
test_set = subset(data_train, split = FALSE)

We define a function about prediction accuracy to compare the target with its proper result

prediction_accuracy = function(pred,actual){
  mean(pred==actual)
}

Here, we use the logistic regression on our 3 variables to optimize the performance

model_log = glm(target~loudness+energy+acousticness, data = training_set)

And then, we used the boosting model. We tried different parameters for the number of trees and the interaction depth to get the best accuracy possible.

library(gbm)
## Warning: package 'gbm' was built under R version 3.6.3
## Loaded gbm 2.1.8
model_boosting = gbm(target~., data = training_set, distribution = "bernoulli", 
                     n.trees = 5000, interaction.depth = 10, shrinkage = 0.01)
pred.boosting = predict(model_boosting, newdata = test_set, type = "response")
## Using 5000 trees...
